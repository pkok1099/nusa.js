Gagal Memuat Game
Pastikan file dijalankan via HTTP server dari folder game/ (bukan file://).
Cara 1: Dari folder game/:
cd game && npx serve .
atau
cd game && python3 -m http.server 8000
Lalu buka http://localhost:3000 atau http://localhost:8000
Cara 2: Gunakan npm:
cd game && npm run dev
Cara 3: Build produksi (tanpa HTTP server):
cd game && npm run build lalu buka dist/index.html di browser
